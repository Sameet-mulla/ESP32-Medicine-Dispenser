# 💊 ESP32 Medicine Dispenser

## 📌 Overview
This is an IoT-based medicine dispenser using ESP32 that automatically dispenses medicines.

## 🚀 Features
- WiFi control
- Servo motor dispensing
- SOS button
- Web interface

## 🛠️ Tech Used
ESP32, Arduino IDE, Embedded C, HTML, CSS
